# UKL-3-November-Nomor-1-Sulit
Program ini digunakan untuk membantu pengguna memasukan dan menghitung nilai rata rata ujian mapel produktif dari sejumlah siswa
Pengguna akan diminta memasukkan jumlah siswa dan nilai nilai ujian mereka satu per satu
Setelah semua nilai dimasukkan, program akan menghitung rata-rata nilai dan menampilkannya
Cara Kerja Program Ini :
Program ini menggunakan Scanner Untuk menerima masukan atau inputan nilai ujian dan jumlah siswa dari pengguna
Program akan meminta pengguna untuk memasukkan jumlah siswa melalui nextInt().
Pengulangan for digunakan untuk mengulang proses input dan penjumlahan nilai berdasarkan jumlah siswa yang dimasukkan pengguna.
Semua nilai dijumlahkan ke dalam variabel totalNilai.
Rata Rata nilai dihitung dengan rumus totalNilai dibagi jumlahSiswa.
